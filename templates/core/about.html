About Image Processing Application

Features

  • Upload images with drag-and-drop interface
  • Apply 6 different image filters
  • Cloud storage with AWS S3
  • Download processed images
  • Browse image gallery

Technology Stack

  • Backend: Django 4.2
  • Image Processing: PIL/Pillow, NumPy
  • Cloud Storage: AWS S3
  • Frontend: Bootstrap 5
  • Database: SQLite (development)

Architecture

This application follows a three-tier architecture:

  • Web Tier: Django templates with responsive UI
  • Application Tier: Django views and image processing
  • Storage Tier: Local storage + AWS S3 cloud storage
